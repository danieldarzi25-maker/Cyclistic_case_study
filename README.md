**Tools used:** Excel, Power Query, PivotTables, data visualization

### Project Overview
This project analyzes 12 months of Cyclistic bike-share data to understand the differences between casual riders and annual members. The goal is to help the marketing team design strategies to convert casual riders into members.

### What I Did
- Cleaned and prepared 12 months of ride data using Excel Power Query  
- Created PivotTables and charts to compare rider behavior  
- Calculated ride lengths and summarized usage patterns by day, month, and time  
- Produced a professional Cleaning/ETL report and Final Report with insights and recommendations
